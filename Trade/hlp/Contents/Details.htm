Поля документа:


  • Поле "Номер документа"
    Поле содержит номер счет-фактуры. Поле заполняется вручную, либо номер автоматически генерируется при сохранении (нажмите на сгенерировать).
  • Поле "Дата документа"
    Указывается дата счет-фактуры. Для удобства используйте календарик.
  • Поле "Дата проводки"
    Поле содержит дату проводки документа. По умолчанию подставляется дата из предыдущего поля.
  • Поле "Основание"
    Поле служит для выбора документа основания для счет-фактуры (тип ТТН, Акт выполненных работ, услуг), при выборе автозаполняются поля, заполненные в документе основании.
  • Поле "Валюта"
    Указывается валюта счет-фактуры.Валюта взаиморасчетов фильтрует Р/счета.
  • Поле "Продавец"
    Служит для выбора лица Продавца. При вводе названия лица с клавиатуры и нажатии enter, происходит контекстный поиск лица в справочнике, и выдаются подходящие варианты. Если не найдено ни одного лица, программа предложит расширенный поиск по справочнику лиц, или создать лицо. Галочка открывает доступ к полям Продавца. При выборе Продавца данные поля заполняются автоматически данными из справочника лиц:
  • Поле Название Продавца
    Содержит название лица.
  • ИНН Продавца
    Содержит ИНН лица, поле доступно для редактирования.
  • КПП Продавца
    Содержит КПП лица, поле доступно для редактирования.
  • Адрес Продавца
    Автозаполняется в случае, если определен юридический адрес, иначе, нажатие на кнопку приводит к форме выбора адреса из доступных адресов. Можно ввести адрес вручную.
  • Поле покупатель
    Поля Покупателя заполняются аналогично полям Продавца, и носят тот же смысл.
  • Поле "Договор"
    Поле содержит название (номер договора) между Покупателем и Продавцом. Выбор договора осуществляется нажатием «…» из списка договоров, с типом «Договор (с уточнениями)». Если предварительно заполнены поля «Продавец» или «Покупатель», то поиск договора производится только по указанным лицам, в случае наличия единственного договора, поле заполоняется автоматически. Есть возможность ввести в строку Договор код или номер договора, и если такой документ найден, то поле заполняется. Если поля «Продавец» или «Покупатель» не заполнены, то при выборе Договора произойдет автоподстановка в эти поля значений из договора. Если у договора есть не аннулированные изображения с факсимиле, то слева от контрола выбора договора рисуется иконка штампа. Если навести мышкой на иконку, то появляется список, в котором собраны все НЕ ПОВТОРЯЮЩИЕСЯ штампы, со всех НЕ АННУЛИРОВАННЫХ изображений. Если штампы повторяются, то будет только одна запись каждого штампа. Если изображение аннулировано, то иконка штампа не рисуется, тултипа нет.
  • Поле "Приложение"
    Поле служит для выбора приложения к договору. Выбор документа при нажатии на «…» происходит среди документов с типом «приложение к договору». Если предварительно выбран «Продавец» или «Покупатель», то поиск документа происходит по этим лицам. Приложение выбирается только среди Приложений к выбранному договору (при заполненном поле Договор). Есть возможность ввести в строку Приложение к договору код или номер приложения, и если такой документ найден, то поле заполняется. Если поле приложение заполнено, то договор выбирается автоматически (при незаполненном поле Договор). В случае наличия у договора единственного приложения, поле приложение заполняется автоматически при выборе договора.
  • Поля грузоотправителя
    Поле грузоотправитель служит для выбора лица Грузоотправителя. При вводе названия лица с клавиатуры и нажатии enter, происходит контекстный поиск лица в справочнике, и выдаются подходящие варианты. Если не найдено ни одного лица, программа предложит расширенный поиск по справочнику лиц, или создать лицо. Галочка открывает доступ к полям Грузоотправителя. При выборе Грузоотправителя данные поля заполняются автоматически данными из справочника лиц: Реквизиты Грузоотправителя – поле содержит строку юридических реквизитов организации Грузоотправителя, включая название организации, почтовый индекс, юридический адрес.
  • Поля грузополучателя
    Поля грузополучателя заполняются аналогично полям Грузоотправителя, и носят тот же смысл.
  • Поле "Счет на предоплату"
    Поле служит для выбора счета на предоплату. Выбор документа при нажатии на «…» происходит среди документов с типом «счет на предоплату». Если предварительно выбран «Продавец» или «Покупатель», то поиск документа происходит по этим лицам. Есть возможность ввести в строку код или номер счета, и если такой документ найден, то поле заполняется. В случае наличия платежных документов по выбранному счету на предоплату, они автоматически подставятся в поле Платежные документы. Можно выбрать несколько счетов на предоплату для данной счет-фактуры.
  • Поле "Платежные документы"
    Поле служит для выбора платежных документов. Выбор документа при нажатии на «…» происходит среди документов с типом «платежное поручение или SWIFT». Если предварительно выбран «Продавец» или «Покупатель», то поиск документа происходит по этим лицам, но условие поиска по лицам можно поменять. Есть возможность ввести в строку код или номер счета, и если такой документ найден, то поле заполняется. В случае наличия связанного с документом оплаты счета на предоплату, данный счет проставляется в предыдущее поле. Можно выбрать платежных документов для данной счет-фактуры.
  • Поле "Примечание"
    Тестовое поле – служит для ввода примечания.
  • Позиции по документу основанию
    Табличная часть, содержащая информацию о продуктах и услугах. Данные переносятся из документа основания. В случае создания счета-фактуры без документа основания, позиции по документу добавить не возможно.
